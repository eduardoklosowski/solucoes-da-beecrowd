# 1466 - Percurso em Árvore por Nível

## [Descrição](https://www.beecrowd.com.br/judge/pt/problems/view/1466)

## Solução

Primeiramente, precisa-se montar uma [árvore binária de busca](../../../base-teorica/grafos/arvore-binaria-de-busca/README.md) com os valores apresentados e depois para exibir a árvore, pode-se fazer uma [busca em largura](../../../base-teorica/grafos/percursos/README.md#busca-em-largura) para apresentar os elementos na ordem correta.
