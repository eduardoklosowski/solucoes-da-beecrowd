# 1552 - Resgate em Queda Livre

## [Descrição](https://www.beecrowd.com.br/judge/pt/problems/view/1552)

## Solução

O grafo desse exercício é sempre completo, ele tem todas as arestas entre todos os vértices com peso igual à distância entre os pontos. Após montar o grafo, você então precisa obter a [árvore geradora mínima](../../../base-teorica/grafos/arvore-geradora-minima/README.md) para ver qual o mínimo de teia necessário para unir todos os vértices.
