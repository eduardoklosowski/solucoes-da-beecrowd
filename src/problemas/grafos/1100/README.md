# 1100 - Movimentos do Cavalo

## [Descrição](https://www.beecrowd.com.br/judge/pt/problems/view/1100)

## Solução

O algoritmo de [busca em largura](../../../base-teorica/grafos/percursos/README.md#busca-em-largura) oferece uma garantia teórica de obter uma solução com o menor número de passos possível, então é sempre uma ideia a ser aplicada quando a possível resposta não é muito grande.
